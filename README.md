# WanderLust – Airbnb Clone

A full-stack Airbnb-inspired web application where users can browse, create, edit, review, and manage property listings.

## Live Demo

https://wanderluste-major-project.onrender.com/listings

## GitHub Repository

https://github.com/sunny23042005/wanderlustr-major-project

## Features

* User Authentication (Register/Login/Logout)
* Create, Edit and Delete Listings
* Upload Property Images
* Cloud Image Storage using Cloudinary
* Property Categories
* Search Listings
* Reviews and Ratings
* Responsive Design
* Session-based Authentication
* Flash Messages
* Server-side Validation
* MongoDB Database Integration
